# AI-Mind Finger Drawing（腦心畫）V1.1

A privacy-aware, mobile-first five-second finger drawing experience for GitHub Pages.

## V1.1 improvements

- The **Start** button is now placed above the canvas and remains visible in the first mobile screen.
- More compact mobile header, title, instructions, timer, and color palette.
- Full **中文 / English** language switch, including generated mood descriptions and analysis labels.
- Language preference is stored only in the browser with `localStorage`.
- Drawing, analysis, and artwork generation remain fully local in the browser.
- No account, database, cookies, analytics, visitor counter, or external data transmission.

## Files

- `index.html`
- `style.css`
- `app.js`
- `.nojekyll`

## Update the existing GitHub repository

1. Unzip this package.
2. In the existing `ai-mind-finger-drawing` repository, replace `index.html`, `style.css`, `app.js`, `README.md`, and `.nojekyll` with these V1.1 files.
3. Commit directly to the `main` branch.
4. Wait for the GitHub Pages workflow to finish.
5. Refresh the published site. A hard refresh may be needed because the browser can cache older CSS and JavaScript.

## Privacy and interpretation

This is an artistic self-awareness experience, not a psychological, medical, or health diagnosis. When variable touch pressure is unavailable, the interface clearly states that contact area and line variation are used as supporting estimates.
