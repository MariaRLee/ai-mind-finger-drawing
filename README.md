# AI-Mind Care: Figure Drawing（腦心畫）V2.2

A privacy-friendly browser experience for creative self-awareness.

## V2.2 updates

- Removed **Share Artwork / 分享作品** and all Web Share API code.
- Kept **Download Artwork / 下載作品** and **Download Raw Drawing / 下載原始畫作**.
- Renamed **Finish Artwork / 完成作品** to **Analyze Artwork / 分析作品**.
- On mobile, the post-timer action panel stays near the bottom of the screen so these three actions remain visible:
  - **Keep Drawing / 繼續畫畫**
  - **Analyze Artwork / 分析作品**
  - **Start Over / 重新開始**
- Shortened the Chinese introduction and keeps it on one compact line on mobile.
- Retains the compact result view and redesigned desktop workspace from V1.5.

## Main functions

- Choose a 5- or 10-second analysis period.
- Continue drawing freely after the timer ends.
- Analyze the timed drawing features to create mood-inspired art.
- Download the generated artwork.
- Download the original finger drawing.
- Switch between Chinese and English.
- Drawing and analysis remain local to the browser.

## Interpretation boundary

The 5- and 10-second choices are creative interaction options, not scientifically validated psychological measurement durations. Results are for creative expression and self-reflection only, not psychological or medical assessment or diagnosis.
