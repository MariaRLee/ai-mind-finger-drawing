# AI-Mind Finger Drawing（腦心畫）V1.4

AI-Mind Finger Drawing is a lightweight, privacy-friendly browser experience for **creative self-awareness**.

## New in V1.4

1. **Timed analysis + free continuation**
   - Choose **5 seconds** or **10 seconds**.
   - The first timed period is used for line-feature analysis.
   - After the timer ends, the user can:
     - **繼續畫畫 / Keep Drawing**
     - **完成作品 / Finish Artwork**
     - **重新開始 / Start Over**

2. **Save the raw drawing**
   - Users can now save the original hand-drawn image.
   - The system also still allows saving the generated mood-inspired artwork.

3. **Privacy-first**
   - All drawing and analysis stay in the local browser.
   - No analytics, no tracking, no external upload.

- Folder: **/ (root)**

## Notes

- **5 / 10 seconds** are **creative interaction choices**, not scientifically validated psychological measurement durations.
- Output is for **creative expression and self-reflection only**, not psychological or medical diagnosis.
